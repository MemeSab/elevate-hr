## What I did

- Designed and built a task prioritisation app using React and Firebase  
- Structured task logic across urgency/importance quadrants  
- Implemented drag-and-drop functionality for task management  
- Integrated Firebase for real-time data persistence  
- Explored integrations such as Google Calendar to understand end-to-end data flow  

## How I approached it

- Broke down the product into clear components and data structures  
- Focused on usability and simple task interaction  
- Iterated on features like filtering, labels and due dates  
- Treated it like a real product build rather than just a code exercise  

## Why this matters

This project reflects how I think about building and delivering products — structuring ideas, understanding data flow, and turning requirements into working functionality.
